### :rocket: Getting Started

### Client and Server 

Install dependencies

```bash
npm i && cd fp-client && npm i && cd ../fp-server && npm i
```

Set Up __.env__

```bash
cp fp-server/.env.example fp-api/.env
```

Run project 
```bash
npm run dev:client   
npm run dev:server 
```
